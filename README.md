# Help-desk.py
This is a ticketing help desk Python app with features that create support tickets, auto-generate ticket IDs, set priority levels, view all tickets, update ticket status, persist data using a file (JSON), and input validation, Object-Oriented Design (OOP).
Designed and implemented a Python-based IT Help Desk system to manage support tickets.
Applied object-oriented programming, file persistence using JSON, and input validation.
Implemented ticket lifecycle management including creation, status updates, and prioritisation.
Technologies: Python, OOP, JSON, File Handling.
